## Image Specifications

- **Format:** WebP (best compression for web)
- **Dimensions:** 800x600px (4:3 aspect ratio)
- **File Size:** < 100KB per image
- **Background:** White or transparent
- **Quality:** High enough to show product details

## Fallback Strategy

If real product images cannot be obtained:

1. **Use stock photos** from sites like:
   - Unsplash (free, high quality)
   - Pexels (free)
   - Pixabay (free)
   - Search terms: "HDPE pipe", "corrugated pipe", "garden hose", etc.

2. **Create technical illustrations:**
   - Cross-section diagrams
   - Technical drawings
   - 3D renders

3. **Contact partners directly:**
   - Request product photography
   - Ask for marketing materials
   - Request permission to use catalog images

## Current Status

All products currently use fallback gradient backgrounds with category icons. Priority is to add real product images to improve visual appeal and help customers identify products.
