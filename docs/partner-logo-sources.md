## Instructions for Logo Extraction

1. Visit each partner website
2. Right-click on logo → "Inspect Element"
3. Find the image source URL
4. Download the highest quality version (SVG preferred)
5. If only raster available, download largest size
6. Optimize for web:
   - SVG: Clean up unnecessary code
   - PNG: Compress and ensure transparent background
   - Target size: ~200x60px display size

## Manual Action Required

**Since I cannot directly download images from websites, please:**

1. Visit each partner website listed above
2. Download their official logo (usually in header/footer)
3. Save to: `/public/partners/[partner-id]/logo.svg` or `logo.png`
4. Ensure transparent backgrounds for PNG files
5. Optimize file sizes (use tools like SVGO for SVG, TinyPNG for PNG)

**Alternative:** Contact each partner directly to request their official logo files and brand guidelines.
