## Sourcing rules

1. **Manufacturer images first** — Konti Hidroplast, FITT, Plastika KS,
   Plastika NV, Ferplast, Polins, Roto, Palaplast, Perplast, SEL, Confort,
   Xier images are pulled directly from the partner sites and committed
   into `public/products/`.
2. **Royalty-free fallback (Pexels)** — used for three abstract category
   shots that have no client-supplied photo: `damp-proof-membrane.jpg`,
   `recycled-pe-granulates.jpg`, `food-grade-barrel.jpg`. Pexels is
   royalty-free with no attribution requirement.
3. **No hot-linking** — `image:` fields in
   `src/lib/products-data.ts` only reference local paths; remote
   patterns in `next.config.ts` are kept for legacy compatibility but no
   longer used by the catalog.
4. **Premium fallback artwork** — when an image fails to load, the
   `ProductGroupCard` renders a custom dark-glass card with the category
   icon and label rather than a generic placeholder.
